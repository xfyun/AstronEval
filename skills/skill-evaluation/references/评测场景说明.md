# 评测场景说明

本文档定义 5 种评测场景的含义、适用情况，以及与 Skill / 驱动模型 / 运行框架数量的对应关系。

---

## 场景速查

| 场景 | 适用情况 | Skill 数量 | 驱动模型数量 | 运行框架数量 |
|------|----------|-----------|-------------|-------------|
| 快速效果验证 | 想快速看一个 Skill 在某个模型下能不能跑、效果大致如何 | 1 个 | 1 个 | 1 个（默认） |
| 同类 Skill 横评 | 多个功能相似的 Skill，想选出最合适的一个 | 多个 | 1 个 | 1 个（默认） |
| 驱动模型横评 | Skill 已固定，想看哪个模型驱动它效果最好 | 1 个 | 多个 | 1 个（默认） |
| Skill 版本对比、A/B Test | 同一个 Skill 不同版本，比较迭代效果 | 多个（同 Skill 不同版本） | 1 个 | 1 个（默认） |
| 运行框架横评 | 同一 Skill 在不同运行框架下对比表现 | 1 个 | 1 个 | 多个 |

---

## 场景详细

### 1. 快速效果验证

**典型用户语句**：

- "我想评测下 xxx Skill"
- "看看 xxx Skill 跑得怎么样"
- "xxx Skill 在 GLM-5 上效果如何"

**配置策略**：

| 项 | 值 |
|----|----|
| Skill | 1 个 |
| 驱动模型 | 1 个 |
| 评委模型 | 1 个（系统自动推荐一个） |

**场景目标**：验证 Skill 是否可执行，给出基础得分。

---

### 2. 同类 Skill 横评

**典型用户语句**：

- "对比一下这几个测试用例生成 Skill 哪个好"
- "在 GLM-5 上比较 skill-A 和 skill-B"

**配置策略**：

| 项 | 值 |
|----|----|
| Skill | 多个（功能相似） |
| 驱动模型 | 1 个（统一驱动） |
| 评委模型 | 1 个 |

**场景目标**：在同一驱动模型下选出最合适的 Skill。

---

### 3. 驱动模型横评

**典型用户语句**：

- "我想知道哪个模型执行 xxx Skill 效果更好"
- "xxx Skill 在 GLM-5、DeepSeek-V3 上对比一下"

**配置策略**：

| 项 | 值 |
|----|----|
| Skill | 1 个（固定） |
| 驱动模型 | 多个 |
| 评委模型 | 1 个（建议与驱动模型不重叠） |

**场景目标**：找出最适合驱动该 Skill 的模型。

**特殊提示**：若评委模型与驱动模型有重叠，必须告知用户：
> ⚠️ 驱动模型中包含评委模型「{judge-display-name}」，评委对自身输出评分可能存在自我偏好偏差，影响客观性。

---

### 4. Skill 版本对比、A/B Test

**典型用户语句**：

- "对比一下 v1 和 v2 两个版本的 xxx Skill"
- "新版本 xxx Skill 效果有提升吗"

**配置策略**：

| 项 | 值 |
|----|----|
| Skill | 多个（同 Skill 不同版本，作为多个评测对象上传） | 
| 驱动模型 | 1 个（统一） |
| 评委模型 | 1 个 |

**场景目标**：评估 Skill 版本迭代是否带来提升。

**注意事项**：

| 注意 | 说明 |
|------|------|
| Skill 名称建议加版本号 | 让最近任务列表清晰可读 |
| 用同一驱动模型 | 控制变量，便于对比 |

---

### 5. 运行框架横评

**典型用户语句**：

- "xxx Skill 在 claude-code 和 openclaw 上对比一下"
- "同一个 Skill 用不同框架跑效果一样吗"
- "哪个运行框架更适合 xxx Skill"

**配置策略**：

| 项 | 值 |
|----|----|
| Skill | 1 个（固定） |
| 驱动模型 | 1 个（统一） |
| 运行框架 | 多个 |
| 评委模型 | 1 个 |

**场景目标**：找出最适合运行该 Skill 的框架。

---

## 推荐选项映射

阶段1 任务3 步骤3 中，向用户展示的 5 个推荐选项与本文档场景的对应关系：

| 推荐选项 | 评测场景 |
|----------|----------|
| 1 | 快速效果验证 |
| 2 | 同类 Skill 横评 |
| 3 | 驱动模型横评 |
| 4 | 运行框架横评 |
| 5 | 不确定（进入场景引导） |

进入"场景引导"后，向用户展示 5 个细粒度选项：

| 引导选项 | 评测场景 |
|----------|----------|
| 1 | 快速效果验证 |
| 2 | 同类 Skill 横评 |
| 3 | 驱动模型横评 |
| 4 | Skill 版本对比、A/B Test |
| 5 | 运行框架横评 |

## 场景驱动的下游决策

| 阶段 | 步骤 | 影响 |
|------|------|------|
| 阶段2 任务1 | 选择 Skill | 数量决定单选/多选交互 |
| 阶段2 任务2 | 选择运行框架 | 数量决定单选/多选交互 |
| 阶段2 任务3 | 选择驱动模型 | 数量决定单选/多选交互 |
| 阶段2 任务4 | 选择评委模型 | 始终单选；驱动模型横评时检查重叠 |
