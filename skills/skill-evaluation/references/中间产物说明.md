# 中间产物说明

本文档定义 Skill 评测流程中产生的中间文件路径与字段。

> ⚠️ 这些文件均为内部实现细节，**不应展示给用户**。文件路径与 `dataset-id`、`task-id` 等内部 ID 仅在脚本调用时使用。

---

## 快速参考

| 阶段 | 文件 | 说明 |
|------|------|------|
| 跨会话 | `auth.json` | 鉴权 Token |
| | `custom-models.json` | 历史自定义模型（按 usage 区分） |
| | `eval-models-cache.json` | 模型列表缓存 |
| | `tasks.json` | 任务索引（含 task_id、状态、提交时间） |
| 配置评测对象 | `eval-runtimes.json` | 运行框架配置 |
| | `eval-judge.json` | 评委配置 |
| | `eval-models.json` | 驱动模型列表 |
| 数据准备 | `eval-dataset.json` | 评测集内容 |
| | `eval-dataset-meta.json` | 评测集元信息（含 dataset-id） |
| | `eval-datamaker.json` | 数据合成任务信息 |
| | `eval-datamaker-result.json` | 数据合成结果 |
| 执行评测 | `eval-task.json` | 评测任务信息（含 task-id） |
| | `eval-result.json` | 评测结果（含报告链接） |

## 跨会话文件

### auth.json

**路径**：`{work-dir}/.eval/auth.json`

存储 OAuth2 Token。失效时通过 `login` 命令刷新。

| 字段 | 类型 | 说明 |
|------|------|------|
| `access_token` | string | 访问令牌 |
| `expires_in` | int | 有效时长（秒） |
| `created_at` | string | 创建时间（ISO 8601） |
| `expires_at` | string | 过期时间（ISO 8601） |

### custom-models.json

**路径**：`{work-dir}/.eval/custom-models.json`

存储用户历史添加的自定义模型，按 `usage` 区分。

| 字段 | 类型 | 说明 |
|------|------|------|
| `usage` | string | `candidate`（驱动）/ `judge`（评委） |
| `model_id` | string | 模型标识 |
| `api_key` | string | API Key |
| `api_url` | string | API URL |

### tasks.json

**路径**：`{work-dir}/.eval/tasks.json`

任务索引，供 `recent-tasks` 命令查询。

| 字段 | 类型 | 说明 |
|------|------|------|
| `session_id` | string | 会话 ID |
| `skill_name` | string | Skill 名称 |
| `model` | string | 驱动模型显示名称 |
| `task_id` | string | 任务 ID（配置中状态为空） |
| `status` | string | `配置中` / `执行中` / `已完成` |
| `progress` | int | 执行进度（0-100） |
| `submit_time` | string | 提交时间 |

## 会话目录文件

会话目录路径：`{work-dir}/.eval/{session-id}/`

`{session-id}` 命名规则：`YYYYMMDD_HHMMSS_{skill-name}`。

### eval-runtimes.json

运行框架对象数组：

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | string | 框架类型：`claude-code` / `openclaw` / `hermes` |

### eval-judge.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 评委模型 ID |
| `model` | string | 模型标识 |
| `display_name` | string | 显示名称 |
| `source` | string | `limited_free` / `custom` |
| `recommend_model_id` | string | TokenPlan/EvalPlan 推荐 ID（仅 `limited_free`） |

### eval-models.json

驱动模型对象数组，结构同 `eval-judge.json`，可包含多个对象。

### eval-dataset.json

评测集用例列表（JSON 数组）。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `case_id` | string | 是 | 用例唯一标识 |
| `question` | string | 是 | 评测问题 |
| `category` | string | 否 | 问题分类 |
| `keypoint` | string | 否 | 评测要点 |
| `filelist` | array<string> | 否 | 关联文件路径 |

### eval-dataset-meta.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `dataset_id` | string | 评测集 ID |
| `case_count` | int | 用例总数 |
| `uploaded_at` | string | 上传时间 |

### eval-datamaker.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `datamaker_id` | string | 合成任务 ID |
| `submitted_at` | string | 提交时间 |

### eval-datamaker-result.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `status` | string | `Pending` / `Running` / `Succeeded` / `Failed` |
| `dataset_id` | string | 合成完成后的评测集 ID |
| `error` | string | 失败原因（仅 Failed） |

### eval-task.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `task_id` | string | 评测任务 ID |
| `submitted_at` | string | 提交时间 |
| `dataset_id` | string | 评测集 ID |
| `skill_urls` | array<string> | Skill zip URL 列表 |
| `runtimes` | array<object> | 运行框架配置 |
| `runtimes[].type` | string | 框架类型 |

### eval-result.json

| 字段 | 类型 | 说明 |
|------|------|------|
| `status` | string | `Pending` / `Running` / `Succeeded` / `Failed` |
| `progress` | int | 执行进度（0-100） |
| `pass_rate` | float | 总体通过率（仅 Succeeded） |
| `model_scores` | object | 各模型得分（仅 Succeeded） |
| `runtime_scores` | object | 各运行框架得分（仅多框架横评且 Succeeded） |
| `artifacts` | object | 报告产物，含 `report_render_url` |
| `error` | string | 失败原因（仅 Failed） |

## 暴露给用户的字段

下表是允许在与用户交互时展示的字段（其他字段必须保持内部）：

| 文件 | 可展示字段 | 不可展示字段 |
|------|------------|--------------|
| `eval-runtimes.json` | 框架类型 | - |
| `eval-dataset.json` | `case_id`、`question`（限 50 字符摘要） | `keypoint`、`filelist`、其他详情 |
| `eval-dataset-meta.json` | `case_count` | `dataset_id` |
| `eval-task.json` | - | `task_id`（仅在最近任务列表中可见） |
| `eval-result.json` | `pass_rate`、`model_scores`、`runtime_scores`、`artifacts.report_render_url` | `task_id`、`error`（错误时除外） |
| `tasks.json` | `skill_name`、`model`、`status`、`task_id`、`progress`、`submit_time` | `session_id` |
